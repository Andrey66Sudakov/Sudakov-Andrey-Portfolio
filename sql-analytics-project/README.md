# SQL Analytics Project: Отчетность строительной компании

## Описание
Проект выполнен в рамках курса **"SQL и получение данных"** (PostgreSQL, Нетология).

**Задача:** Разработать 10 аналитических SQL-запросов для базы данных строительной компании `stroy`.

## 🎯 Что реализовано

### Бизнес-метрики:
1. Подсчет проектов по годам
2. Расчет возраста и стажа сотрудников
3. Анализ платежей от контрагентов
4. Расчет бонусов руководителей проектов
5. Накопительные итоги платежей
6. Рекурсивный обход организационной структуры
7. Скользящие средние и оконные функции
8. Создание материализованных представлений

## 🛠 Технический стек
- **СУБД:** PostgreSQL 14+
- **Конструкции:** CTE, рекурсивные запросы, оконные функции
- **Объекты:** Materialized Views, агрегатные функции

## 📈 Результат
- **Оценка:** 135 из 150 баллов (зачёт)
- **Запросов выполнено:** 10 из 10
- **Время выполнения:** < 1 сек на запрос
- **Отзыв:** "Поздравляю с хорошо выполненной работой!"
